
**BasiBits** is a fun, beginner-friendly Web3 dApp built on the **Hedera Hashgraph Network**.  
It helps users learn and interact with blockchain technology by sending and receiving simple digital tokens.

---

## Features
- Send and receive HBAR or custom tokens on Hedera testnet  
- Simple, interactive UI for beginners  
- Fast, low-cost transactions using Hedera SDK  

---

## Tech Stack
- Node.js + Express (Backend)
- Hedera JavaScript SDK
- HTML, CSS, JavaScript (Frontend)
- Render (Backend Hosting)
- Netlify (Frontend Hosting)

---

## Setup
```bash
git clone https://github.com/<your-username>/basibits.git
cd basibits
npm install
````

Create a `.env` file:

```
OPERATOR_ID=0.0.xxxxx
OPERATOR_KEY=302e0201...
PORT=4000
```

Run the backend:

```bash
npm start
```

Open `frontend/index.html` in your browser.
